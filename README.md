## 项目简介

本系统是一个功能完备的零食在线销售管理平台，采用前后端分离架构开发。系统涵盖商品展示、购物车、订单管理、用户中心、收藏、评价以及后台管理等核心功能，为零食电商提供一站式解决方案。
